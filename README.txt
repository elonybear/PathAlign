Path Align - The Cooperative Action Puzzle

Requires 2 Xbox 360 Controllers
Player 1 
  Left stick - move
  A - jump
  Right trigger - shoot

Player 2
  Left stick - turn
  X - fill blocks
  Y - destroy last structure (can be used repeatedly)
  D-Pad (Up and Down) - change blocks available

F1 - Invincibility for both players
F2 - All powerups, infinite amount of each

Bugs:
  - Sprites on various powerups can be facing the wrong way
    depending on the direction the ship is moving
  - Ship can sometimes glitch through walls
  - Player can get stuck in Ship's blocks
      - Ship can destroy blocks and free player
